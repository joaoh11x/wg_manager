# Avatar Padrão

Coloque sua imagem de avatar padrão aqui com o nome `default_avatar.png`.

Esta imagem será usada como foto de perfil para todos os usuários que ainda não fizeram upload de uma foto própria.

## Requisitos:
- Nome do arquivo: `default_avatar.png`
- Formato: PNG, JPG, JPEG ou GIF
- Tamanho recomendado: qualquer tamanho (será redimensionada automaticamente para 200x200 pixels)
- A imagem será convertida para PNG automaticamente

## Como usar:
1. Adicione sua imagem neste diretório com o nome `default_avatar.png`
2. Reinicie a aplicação (se necessário)
3. Todos os usuários sem avatar personalizado verão esta imagem
